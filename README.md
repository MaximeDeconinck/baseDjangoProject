# baseDjangoProject

Base main-app Django project with basic login and register features.

![Capture d’écran 2024-01-28 à 18 49 03](https://github.com/MaximeDeconinck/baseDjangoProject/assets/83770758/0a89166f-cd66-4413-baab-c01ad6596939)

## Installation

The only library needed is django-admin :

```bash
pip install django-admin
```

## Usage

```
python3 manage.py runserver
```
